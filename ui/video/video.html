Video and Audio Settings

The PixC4-Jetson provides multiple video and audio streams. You can change the parameters for each below. To disable a stream, enter 0 for the birate.

Primary Video Stream Encoding

This stream is used for video over the primary low-latency and high-bandwdith radio link. Typical bitrates are 1000-4000 kpbs.

Backup MAVPN Video Stream Encoding

This stream is used for video over the secondary link via the MAVPN connection, which is typically a lower-bandwith link like cellular. Typical bitrates are 800-2000 kpbs.

ATAK Stream Encoding

This stream is used for video to the TAK network. These streams are typically used on small screens. Typical bitrates are 500-1000 kpbs.

MAVNet Video Server Stream Encoding

This stream is sent to the MAVNet cloud video distribution server and display on the webGCS user interfaces. It uses the link which provides an Internet route (typically cellular). Consider that total cellular bandwidth usage is this stream combined with the MAVPN stream. Typical bitrates are 500-1500 kbps.

Audio Stream Encoding

The audio encoding information below is used for both the primary link and the backup link. Typical bitrate is 128 kbps.
