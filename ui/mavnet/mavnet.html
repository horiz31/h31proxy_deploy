H31 Proxy Settings

The H31 Proxy software provides the network inteface between the FMU and the various network endpoints. It also interacts with the video system to start and stop video streams in certain circumstances.

Primary Telemtry Link Information

This link is the primary telemetry link. Typically multicast on a Mobile Ad Hoc network. These settings must match the GCS being used. Default port is 14550 for primary.

Backup Telemtry Link Information

This link is the backup telemetry link. Typically multicast over cellular using MAVPN. These settings must match the GCS being used. Default port is 14560 for backup.

ATAK CoT Setup

These settings are used for the ATAK Cursor-on-target generator. The mulicast address should match the CoT multicast address defined by the TAK network.

FMU Setup

These settings are used to enable the companion computer to interface with the FMU. These settings must match the FMU firmware settings.
